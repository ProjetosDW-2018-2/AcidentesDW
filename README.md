# Repositorio Acidentes
## Base de dados dos acidentes da polícia federal do brasil

### Como instalar
- Extraia o reposítorio para a pasta htdocs do xampp
- Inicie o serviço do Apache
- Acesse via `localhost/nome-do-diretorio`
- Para acessar com o banco online, encaminhe um e-mail para <gaabriiel.xd@gmail.com> indentificando-se e com seu IPV4 atual, para que possa ser liberado o acesso. (Acessar IP local: https://www.whatismyip.com/)
- Utilize o menu lateral para acessar as consultas disponíveis

Repositório da 1VA:
https://github.com/DW-Acidentes
